# Summary Statistics

Summary statistics for mtDNA transcript abundance derived from GTEx v8. 

Data are available at: https://doi.org/10.6084/m9.figshare.31950315. 

Files are tab-delimited and produced with PLINK. Columns include: GENE (gene symbol), CHR (chromosome), SNP (chromosome, bp, alleles), BP (base pair position), A1 (effect allele), TEST (type of test), NMISS (number of non-missing samples), BETA (effect size), SE (standard error), L95/U95 (95% confidence interval), STAT (test statistic), and P (p-value). 

Only SNPs with P < 0.05 are included and use is subject to the GTEx Data Use Policy.
